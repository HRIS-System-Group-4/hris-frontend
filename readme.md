   ```bash
   git clone https://github.com/HRIS-System-Group-4/hris-frontend
   cd hris-frontend
   ```

2. **Install dependencies**  
   ```bash
   npm install
   ```

3. **Run development server**  
   ```bash
   npm run dev
   ```
